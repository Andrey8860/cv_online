Over the course of my career I've delivered 5 complex projects in different domains. Distributed teams, external compliance certifications, overtimes - we've seen it all.

Steep

As a first project, the whole two-year post-launch period of a popular AA video game STEEP was tested, with over 24 updates and 3 full DLCs released to production, passing external compliance certifications from Sony and Microsoft.

A spiritual successor to the previous project, Riders Republic, took two years to be tested from the prototype stage up until the eventual successful release to public. E-commerce Migration Project

New front-end, new back-end, and brand-new microservices layer in between multiplied by four different websites - 1.5 years of working with talents from all over the world with an excellent release in the end.

API And Test Automation

Another project for the same Canadian retail client, focused exclusively on building a new API layer from scratch. During this project not only I was back to management responsibilities, but I've also gained a practical experience with Web and API test automation in Java.

Banking Mobile Applications
